# 프로젝트: 간단한 퍼즐 게임

## 목표
- 브라우저에서 실행되는 퍼즐 게임

## 기능
- 퍼즐 보드 표시
- 클릭 이벤트 처리
- 클리어 조건 판단
- 4x4 퍼즐로 2개의 동일한 과일 이미지를 맞추는 게임

## 기술
- 단순 HTML, CSS, Javascript
- `index.html` 1개, `css` 폴더, `js` 폴더로 구성

## 대상
- 고급 개발자

## 폴더 구조

```text
puzzle-game/
├─ index.html
├─ css/
│  └─ styles.css
├─ js/
│  ├─ app.js
│  ├─ board.js
│  ├─ game.js
│  └─ utils.js
├─ assets/
│  └─ images/
│     └─ fruits/
└─ PRD.md
```
